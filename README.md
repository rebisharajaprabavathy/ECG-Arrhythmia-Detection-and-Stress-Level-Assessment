# ECG Arrhythmia Detection and Stress Level Assessment

## 📌 Project Overview
This project presents a multimodal deep learning framework for detecting ECG arrhythmia and assessing stress levels. It combines physiological signal analysis with machine learning techniques to improve heart health prediction.

## 🎯 Objectives
- Detect ECG abnormalities (arrhythmia)
- Predict stress levels using ML/DL models
- Assist in early heart disease risk identification

## 🛠️ Technologies Used
- Python
- TensorFlow / Keras
- Scikit-learn
- NumPy, Pandas
- Matplotlib / Seaborn
- Google Colab

## 📂 Project Structure
- `newmodel_heart1something (1).ipynb` → Main model notebook

## 📊 Features
- ECG signal preprocessing
- Feature extraction
- Model training and evaluation
- Stress prediction system

## ▶️ How to Run
1. Open the notebook in Google Colab
2. Upload dataset
3. Run all cells
